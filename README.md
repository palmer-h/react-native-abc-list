# react-native-abc-list

A React Native list component with an interactive A to Z sidebar

## Installation

```sh
npm install react-native-abc-list
```

## Usage

```js
import { multiply } from 'react-native-abc-list';

// ...

const result = await multiply(3, 7);
```
